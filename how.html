HOW TO PLAY

>>> Input your name

>>> Input the level you want to play, each level is associated with default speed Note that Beginner/Practice level scores are not stored

>>> input the number on the color you want to play in the box, you can only choose one number

>>> Tap the start button to start the game, the moment the game start the purse button will be visible

>>> The game comprises boxes of many colors that represent number 1 to 9 each. select the number you want to Touch/Tap as many time as possible before the number change location to another box.

>>> If you touch wrong number you loose one life, i f you tap/touch the number you selected your life increase by one life

>>> You can Pause/End the game any time
